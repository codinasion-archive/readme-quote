## Usage

```md
![readme-quote](https://readme-quote.vercel.app/api)
```

![readme-quote](https://readme-quote.vercel.app/api)

#### Text Only

```
https://readme-quote.vercel.app/api?mode=text
```

#### Json Only

```
https://readme-quote.vercel.app/api?mode=json
```

#### Get Quote for a specific category

```md
![readme-quote](https://readme-quote.vercel.app/api?category=motivation)
```

#### Combine mode and category

```
https://readme-quote.vercel.app/api?mode=text&category=motivation
```

```
https://readme-quote.vercel.app/api?mode=json&category=motivation
```
